# Stock Market Data Engineering Pipeline

A beginner-friendly, professional end-to-end data engineering project that fetches stock market data from a REST API, processes it, stores it in an SQLite database, and runs analytical SQL queries.

## Project Architecture
1. **Extract**: Fetches daily stock data (IBM) from the Alpha Vantage REST API in JSON format using the `requests` library.
2. **Transform**: Parses nested JSON into a structured, tabular `pandas.DataFrame`, cleans column names, enforces datatypes, and drops missing values.
3. **Load**: Loads the cleaned dataframe into a local SQLite database (`stock_data.db`) via `pandas.to_sql`.
4. **Analyze**: Connects to SQLite and executes 5 analytical SQL queries (including aggregations and window functions), saving outputs to `data/outputs/*.csv`.

## Folder Structure
```text
stock_pipeline/
├── data/outputs/         # CSV reports generated by analysis
├── src/
│   ├── extract.py        # API Integration
│   ├── transform.py      # Pandas data cleaning
│   ├── load.py           # SQLite database loading
│   └── analyze.py        # SQL Analytics & CSV export

```

## Setup & Execution

### 1. Requirements
Ensure you have Python 3 installed. Install the minimal dependencies:
```bash
pip install -r requirements.txt
```

### 2. Run Pipeline
Execute the main orchestrator script:
```bash
python main.py
```

### 3. Review Outputs
The pipeline will generate a local database file `stock_data.db` and output analytical reports to the `data/outputs/` directory:
- `1_basic_aggregation.csv`: Max, min, and average closing prices.
- `2_monthly_avg_volume.csv`: Average trading volume by month.
- `3_up_vs_down_days.csv`: Counts of days where the stock price went up vs down.
- `4_moving_average.csv`: 7-day moving average of closing prices (uses `AVG() OVER(...)`).
- `5_top_volume_days.csv`: Top 5 trading days by volume (uses `RANK() OVER(...)`).
